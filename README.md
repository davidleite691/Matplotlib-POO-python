## Visualização de Dados com Matplotlib

Este projeto explora a utilização da biblioteca Matplotlib no Python para criação de gráficos e visualização de dados, aplicando diferentes técnicas para representar múltiplas informações de forma clara e organizada.

Durante o desenvolvimento, foram utilizados gráficos com mais de um eixo para exibir diferentes métricas em uma única visualização, além da criação de mosaicos de gráficos para comparação simultânea de dados.

### Objetivos do projeto
- Criar visualizações gráficas com Matplotlib
- Trabalhar com múltiplos eixos em gráficos
- Desenvolver mosaicos e combinações de gráficos
- Comparar diferentes conjuntos de dados visualmente
- Praticar técnicas de visualização de dados com Python

### Tecnologias utilizadas
- Python
- Matplotlib
- NumPy
